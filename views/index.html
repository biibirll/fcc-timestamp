Timestamp Microservice

This is a timestamp microservice designed as a solution to FreeCodeCamp's Back End Development and APIs Certification. The API endpoints are listed in the following table:

API endpoints
Endpoint URL params Status code Response Description
/api/ None 200 {"unix": Number, "utc": String} Returns the current time in unix timestamp and UTC formats in the unix and utc keys respectively.
/api/:date
date
A string representing a date that can be correctly parsed by new Date(date)
 200 {"unix": Number, "utc": String} Returns the given date in unix timestamp and the UTC formats in the unix and utc keys respectively.
/api/:date
date
A string which >can't be parsed correctly by new Date(date)
 400 {"error": "Invalid Date"} Returns an error indicating that the given date couldn't be parsed.
